Activity 1: String Manipulations - Anagram

Go back to list

An anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once. For example, the word anagram can be rearranged into nag a ram, or the word binary into brainy.

  • Create 2 input boxes and a button
  • Upon clicking the button, a message will show whether the words are anagram or not
  • If they are anagram, the message should be green otherwise, red
  • The function must accept values with at least 3 characters
  • Punctuations, special characters, and spaces must be discarded
  • Must not be case sensitive

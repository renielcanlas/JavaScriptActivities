Activity 1: Simple Array

Go back to list
  • Create an empty array
  • Create a input field and a button
  • Upon clicking the button, the value on the input field should be added in the empty array
  • The values of the array must be presented inside a <select> element with attribute multiple
  • The function should not accept blank input
  • The array and list should only contain unique values
  • Create 2 buttons with labels "longest" & "shortest"
  • Upon clicking the longest button, an alert box should show the longest word in the list
  • If there are multiple items with the same length as the longest, the output should be the item added lastly on the list
  • Upon clicking the shortest button, an alert box should show the shortest word in the list
  • If there are multiple items with the same length as the shortest, the output should be the item added first on the list
  • Create a Sort button with label "Sort [a-z]" that will sort alphabetically the contents of the array and the list
  • Upon sorting to [a-z] the label for sort button must be changed to "Sort [z-a] and upon clicking, the sorting must be reversed
  • Create remove button that will remove the selected item from the list and from the array object.
  • The remove must be working with multiple selections
  • Create a clear button that will remove all items from both list and array
  • If there are no entries in the list, other buttons must be disabled
